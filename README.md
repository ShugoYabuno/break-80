# ゴルフ80切り計画

## 目的
最短で80切りを達成する。  
スイング改善より `スコア改善（再現性・ミス削減・リスク管理）` を最優先にする。  
特にドライバーの `左OB（フック）` と `3パット` を最優先で減らす。

## 最重要ルール
- 優先順: `左OB削減 > 3パット削減 > フルショット削減`
- ティーショット: `右OK・左NG`
- 危険ホール: `左が少しでも怖ければ1Wを持たない`
- 球筋: `作らない`（結果としてのストレート〜軽いドローは許容）
- 管理対象: `球筋ではなくフェース`
- 狙い: `常にボギーオン`
- ロングパット: `10m超は1m以内`
- スイング強度: `フルショット禁止、基本7割`
- 練習評価: `ナイスショットではなくミス幅`
- Trackman の見方: `右寄りのデータでも、左が消えているなら許容`

## 成功基準
- OB: `3〜4 -> 1以下`
- 3パット: `3〜5 -> 1以下`
- パーオン率: `12%でも問題なし（ボギーオン前提）`
- ボギーオン率: `42% -> 55%以上`
- ドライバー左ミス: `0を最優先`
- 危険ホールでの1W使用: `0を目指す`

## プレイヤープロファイル
- 最大の敗因は `左OB`。特に `フック・引っかけ` は即崩れにつながる
- `プッシュ` と `フック` が両方出るため、課題は `フェース管理の不安定さ`
- 安全にいこうとすると `当てにいく動き` になり、前傾がほどけて弱いインパクトになりやすい
- このプレイヤーは `ドローで攻める人` ではなく、`捕まえないことで結果的に軽く捕まる人`

## 判断基準
- すべての提案は `左を消せるか` と `スコアが減るか` だけで評価する
- `ナイスショット狙い` `捕まえにいく` `強く振る` は採用しない
- 正しい思考は `左に行かないことが正義` `右ミスOK` `ミス幅を小さくする`

## クイックスタート
1. ラウンド前に `checklists/round-checklist.md` を確認する
2. 練習ごとに `templates/practice-log.md` をコピーして記録する
3. ラウンドごとに `templates/round-log.md` をコピーして記録する
4. 週1で `templates/weekly-review.md` をコピーして改善点を1つだけ決める

## Trackman 連携

### 手動で入れる
- `portal.trackmangolf.com` の `Activities` で `Course / Game / Combine / Report / Screencast` を確認する
- `Practice Sessions` は `Trackman Golf App` 側で確認する
- `Shot Analysis` を表データで残したいときは `TPS` から `CSV` を出す
- 方針は `docs/trackman-import.md` を参照

### 半自動で入れる
- `npm install`
- `npm run trackman:import`
- Chrome で Trackman にログインして、保存したい画面を開く
- Enter を押すと `data/trackman/captures/` にキャプチャと下書きが保存される
- 手順詳細は `docs/trackman-browser-import.md` を参照

### ブラウザ内 JavaScript で抜く
- `npm run trackman:browser-script` で貼り付け用コードを表示する
- Trackman のページで DevTools Console に貼る
- その場で `json` をダウンロードする
- 手順詳細は `docs/trackman-browser-js.md` を参照

## 現在のクラブセット
- 詳細は `docs/club-set.md` を参照
- セッティングの論点: `7W` と `4U` の併用はせず、どちらを入れるか検討中
- ドライバー方針: `短尺44インチ` で再現性優先

## 運用ルール
- キーワード: `低く出す / ボールの先を打つ / フェースを返さない / 前傾キープ / 左主導`
- 難しい技術論は禁止
- 指示はシンプル・即実行のみ
- 1ラウンドで改善できる行動だけ採用する
- ティーショットは `左が少しでも怖ければ1Wを持たない`。`1番手下げる` ではなく、`4H / 7W` まで下げてよい
- 危険ホールでは `打てるか` ではなく `1Wを外せるか` を先に決める
- アイアンは `ストレート〜軽いドローでOK` だが、意識は `低く出す / ボールの先を打つ / フェースを返さない` だけにする
- セカンドは `パーオン狙い禁止・ボギーオン100%` を徹底する。`次の1打を打ちやすい場所` を優先する
- アプローチは `寄せるより乗せる`。ミスの向きは常に `グリーン方向`
- パターは `1パット狙い` より `3パット回避` を優先する。`10m超` は `ショートしない` より `1m以内に残す`
- 練習ログと週次レビューでは `左OBにつながる傾向` を最優先で扱う
