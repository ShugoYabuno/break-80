## 購入・フィッティングメモ

- アイアンは `BLUEPRINT S + DG105 S200` を採用。`AWT 3.0 LITE` もフィッティング結果は良かったが、納期と価格差を考慮して中古の `DG105` 装着モデルを選択
- アイアン検討時の前提:
  - 左ミスを減らしたい
  - ライ角は `トゥーダウンしやすい` 前提で大きくは触らない
  - ドライバーHSは `平均37 / MAX41`、7Iキャリーは `140y前後`
- ドライバーは `ATTAS SUNRISE 40R` の振りやすさは残しつつ、`振った先の左` を減らしたい
- 短尺ドライバーの判断メモ:
  - 44インチの短尺は通常尺より `左に寄りやすく、硬く感じやすい`
  - `SPEEDER NX GREEN 40S` は 44インチのバットカット後にハードに感じた
  - そのため `NX GREEN 40R` を購入し、`左はモデル特性で減らして、硬さでは追い込みすぎない` 方針にする

## 運用メモ
- `7W` にするか `4U` にするかは継続検討

- シャフト未記載のクラブは、分かった時点でこのファイルに追記する
